## Installation

#### Clone the project

```bash
  git clone https://github.com/AliNikandish/FullStack-BlogApp-Nextjs14.git
```

#### Go to the project directory

```bash
  cd FullStack-BlogApp-Nextjs14
```

#### Install BlogApp with npm

```bash
  npm install
```

#### env:
```bash
  add variable to env file
```
#### Run:
```bash
  npm run dev
```
